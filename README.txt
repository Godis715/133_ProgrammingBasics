#133_ProgrammingBasics

Графический редактор. За основу работы взяты примитивы (точки, дуги и отрезки),
составляющиеся в более сложные фигуры путем сопоставления им требований.

Фундаментальные требования:
(Через них выражаются все остальные, более сложные требования)
	1. Расстояние между точками равно R (при R = 0 точки совпадают)
	2. Расстояние от точки до отрезка равно R (при R  = 0 точка принадлежит отрезку)
	3. Точки лежат по одну сторону от прямой
	4. Точка лежит в заданном секторе (отн. центра дуги)
	5. Точка лежит на расстоянии R от центра дуги (невыразимо чеерез (1), т.к. центр дуги не храним)
	6. Прямые прпендикулярны
	
Требуемый функционал программы:
	1. Добавить примитив на сцену
	2. Переместить/повернуть/проскалировать примитив
	3. Выделение нескольких примитивов
	4. Пункт (2) для группы из нескольких примитивов в случае, когда не нарушатся уже существующие требования.
		В противном случае не совершаем это действие (отменяем ранее наложеные требования)
	5. Постановление требования n примитивам (пр.: n отрезков пересекаются в 1 точке)
	6. Удалить примитив со сцены
	7. Удалить существующее требование
	8. Сохранить/загрузить сцену в ее текущем состоянии
